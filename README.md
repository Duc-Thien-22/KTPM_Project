# Event Management System – JavaFX

A desktop application for managing events, developed with JavaFX and MySQL.  
Includes user registration, login, event booking, and notification features.  
Manual test cases and unit tests (JUnit 5 & Mockito) were implemented to ensure product quality.

## Technologies
- JavaFX, MySQL
- JUnit 5, Mockito
- Postman, Git

## How to run
1. Clone this repository
2. Import as Maven project
3. Run `Main.java`
